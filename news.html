The 10th Anniversary of the First Open Source Release - 19 Oct 2023

Today we are celebrating a decade of KeyStore Explorer as open source software. On October 19, 2013 version 5.0.0 of KeyStore Explorer was released under the GPLv3 license on Sourceforge.

The history of KSE up to this date and how it became open source was documented by its creator Wayne in a very captivating manner in his blog. If you haven't read this story yet, here are the links:

A History of KeyStore Explorer – Part One
A History of KeyStore Explorer – Part Two
A History of KeyStore Explorer – Part Three
New Owner Wanted for KeyStore Explorer
KeyStore Explorer - OSS Update
KeyStore Explorer Now Open Source

In those last 10 years more than 100 new features have been added in 18 releases. To name just a few of the most important ones:

  • Support for ECC (Elliptic Curve Cryptography), EdDSA (Ed25519 and Ed448) and RSASSA-PSS signatures
  • Support for MSCAPI and PKCS#11, allowing the use of smart cards and HSMs with KSE
  • System clipboard integration
  • More CA features like extension templates, custom certificate extensions and CRL creation
  • JWT (JSON Web Token) creation

Many of those features were user contributions. All in all almost 30 persons have contributed code or translations to this project and I want to thank all of them for making this project what it is today! The top 5 contributors are:

There has been a constant rise in the user count over the last 10 years as well. In 2013, monthly downloads averaged 1,500, but today, that figure has increased to around 35,000.

What is planned for the future? KSE v5.6.0 will be the next big feature release, and it will focus on the following topics:

  • Improved PKCS#12 support
  • Extended verification features (verifying signatures of jars and CSRs)
  • Extended JWT support (for example key import/export in JWK format)

KSE 5.5.2 Released - 22 Jan 2023

This release contains lots of small improvements and a few bug fixes.

Highlights:

  • Improved key details dialogs with more information about the keys.
  • The subject DN and extensions for a new certificate can now be obtained from another certificate (button "Transfer Name and Extensions").
  • A redesigned preferences dialog.

The complete list of changes can be found in the release notes.

KSE 5.5.1 Released - 27 Jan 2022

Highlights of this release:

  • KSE can now create and sign JWTs (JSON Web Tokens).
  • The French translation has been extended and improved.
  • A bug that made the import of CA replies with EC keys impossible has been fixed.

The complete list of changes can be found in the release notes.

One Million Downloads - 14 Oct 2020

Today KeyStore Explorer has crossed one million downloads on GitHub. This includes all releases since November 2015. Not included are downloads from the time when KSE was hosted on Sourceforge (2013 - 2015) or when KSE was Closed Source (2004 - 2013). It took around three years to reach 500,000 downloads and then only two more years for one million, which shows that the number of users is steadily increasing.

If you are interested in a more detailed breakdown of the download numbers, there is a nice website called GitHub Release Stats.

Online User Manual - 31 Dec 2019

KeyStore Explorer's user manual is now part of this website. Future releases of KeyStore Explorer won't include a user manual anymore. Instead the menu item 'Help' will link to the online user manual.

Please keep in mind that you can contribute to the KeyStore Explorer project by improving the documentation. Like KeyStore Explorer itself this website is managed in a repository on GitHub.
